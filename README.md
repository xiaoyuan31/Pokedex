# Pokedex

A simple Pokédex application built with Kotlin — a catalogue of Pokémon with basic details and browsing features.

This project is a **mobile Pokémon index app** inspired by the classic Pokédex concept: cataloguing Pokémon species with their key data such as names, types, and stats.

## 📌 Features

- 📱 Browse a list of Pokémon  
- 🔎 Search Pokémon by name  
- 🔍 View basic details (type, stats, artwork)  
- 🚀 Built with modern Android tools (Kotlin)

## 🧠 Tech Stack

This project is written in **Kotlin** and structured as an Android app.

- 🧩 Kotlin  
- 📦 Android SDK  
- 📱 Android Activity & Fragments  
- 📁 Organized in `app/` and `gradle/` folders
